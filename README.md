
# Semana 3 – PCA y Clustering

Este directorio contiene los materiales y ejercicios correspondientes a la **Semana 3** del curso *Minería de Datos 2026*.  
El objetivo principal es aplicar **Análisis de Componentes Principales (PCA)** y **Clustering** para segmentar clientes y visualizar patrones.

---

## Contenido

- **Notebook principal:**  
  `AO1_Clase4_Martinez_Lionel.ipynb`  
  Incluye el análisis exploratorio (EDA), aplicación de PCA y segmentación con K-Means.

- **Capturas de gráficos:**  
  - Histograma de Edad  
  - Histograma de Ingresos  
  - Matriz de Correlación (Edad, Ingresos, Experiencia)  
  - Dispersión Edad vs Ingresos  
  - Clusters después de PCA

---

## Resultados destacados

- Las variables Edad, Ingresos y Experiencia presentan correlaciones positivas fuertes.  
- PCA permitió reducir la dimensionalidad y representar los datos en un espacio 2D.  
- El clustering reveló grupos diferenciados de clientes, útiles para estrategias de segmentación.

---

## Conclusión

La combinación de **EDA + PCA + Clustering** ofrece una visión más clara de los perfiles de clientes y facilita la interpretación de patrones.  
Este material servirá como base para la discusión en el foro y para futuras aplicaciones en marketing y análisis de riesgo.
