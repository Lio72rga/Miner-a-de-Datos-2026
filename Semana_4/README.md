# Visualización de Datos - Actividad

## Descripción
En este trabajo se realiza un análisis exploratorio y visualización de datos sobre un dataset de clientes de una tienda online.

## Contenido

- EDA_Clientes: análisis exploratorio (distribuciones, outliers, relaciones)
- Visualización Avanzada: pairplot, heatmap y violin plot

## Herramientas utilizadas
- Python
- Pandas
- Matplotlib
- Seaborn

## Conclusión
Se identificó una relación positiva entre ingresos y cantidad de compras, además de patrones diferenciados según la categoría de cliente.
