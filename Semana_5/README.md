#  Minería de Datos - Clase 5
## Limpieza y Transformación de Datos

Este repositorio contiene el desarrollo práctico de la Clase 5 de Minería de Datos, enfocado en técnicas de limpieza, transformación y preprocesamiento de datos utilizando Python.

---

#  Objetivos de la Actividad

- Detectar y manejar valores faltantes.
- Identificar y tratar outliers.
- Aplicar normalización y estandarización.
- Transformar variables categóricas.
- Comprender la importancia del preprocesamiento en Machine Learning.

#  Actividades Realizadas

## 1 Manejo de Datos Faltantes
- Identificación de valores nulos.
- Eliminación de filas con datos faltantes.
- Imputación utilizando la moda.

## 2 Detección y Tratamiento de Outliers
- Visualización mediante boxplots.
- Aplicación del método IQR.
- Eliminación de valores atípicos.

## 3 Transformación de Datos
### Normalización
Aplicación de Min-Max Scaling.

### Estandarización
Aplicación de Z-score Scaling.

### Variables Categóricas
- One-Hot Encoding
- Label Encoding
